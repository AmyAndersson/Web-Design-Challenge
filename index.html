Summary Latittude vs X

This project looked at the weather in at least 500 countries to determine weather changes and their relation to the equator. In order to do this data was pulled from the OpenWeatherMaps website using an API.

Once the data was acquired Matplotlib and other visulisation libraries were used to plot and view the weather data. Analysis and Visulisations include temperature, humidity, cloudiness and windspeed data. Results from this analysis are displayed on this site!
